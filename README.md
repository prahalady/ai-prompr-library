# 교원 AI 프롬프트 라이브러리 (직업계고 교원연수)

2026 전국 현장실습·취업 교육 역량강화 연수용 프롬프트 모음입니다.
교안(「2026 교원 직무역량 향상 연수 — AI 활용 방법」, 전경아)에서 추출한 **원본 프롬프트 22종**과
직업계고 현장에 맞춘 **확장 케이스 19종**, 총 **41개**를 한 페이지에 담았습니다.

## 사용법

`AI프롬프트_라이브러리_직업계고교원연수.html` 파일을 브라우저로 열면 됩니다.

- 각 카드의 **복사** 버튼 → ChatGPT/Gemini/Claude에 붙여넣기
- 대괄호 `[ ]` 부분만 내 상황에 맞게 바꿔 사용
- 상단 **검색창**과 **카테고리 필터**로 원하는 프롬프트를 바로 찾기

| 영역 | 내용 |
|------|------|

| 생기부 | 문장 후보 생성, 다듬기, 점검, 세특·행특, 비식별 처리 |

| 챗봇·재사용 | Gemini Gem / ChatGPT GPTs 지시문 템플릿 |

## ⚠️ 안전 원칙

- 학생 실명·생년월일·연락처·성적 원자료·상담/건강/가정 정보는 **절대 입력 금지**
- "학생 A, 기계과 2학년 학생"처럼 **익명화**, 실습은 **가상 사례**로
- AI 결과는 항상 **초안**이며 최종 판단·수정은 교사가 합니다
